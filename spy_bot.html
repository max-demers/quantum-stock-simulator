L'Approche Quantique

Les modèles classiques échouent à prédire les krachs et les sauts de volatilité. Ce rapport explore comment l'équation de Schrödinger et l'effet tunnel peuvent modéliser la dynamique du S&P 500 (SPY) pour anticiper les ruptures de marché.

Comparaison de Modèles

Le modèle quantique capture mieux les "queues épaisses" (événements extrêmes) que la distribution normale classique.

Le Formalisme de Schrödinger

En finance quantique, le prix d'un actif n'est pas un point précis, mais une fonction d'onde de probabilité $\psi(x,t)$.

iℏ ∂ψ/∂t = [ -ℏ²/2m ∂²/∂x² + V(x) ] ψ

  • x : Le prix (souvent log-prix).
  • m (Masse) : Inversement proportionnelle à la volatilité ($\sigma$). Une forte volatilité = particule légère (bouge vite).
  • V(x) (Potentiel) : Les forces du marché (support, résistance, sentiment).
Insight Clé : Contrairement à la marche aléatoire, l'équation de Schrödinger introduit une dynamique d'interférence et de conservation de l'information.

L'Effet Tunnel : Prédire les Breakouts

En mécanique classique, une bille ne peut pas traverser un mur plus haut que son énergie. En quantique, il existe une probabilité non nulle qu'elle traverse : c'est l'effet tunnel. En trading, cela correspond à un prix traversant un niveau de résistance solide.

  • ✓ Normalisation : Les prix doivent être mappés sur une grille spatiale discrète $x$ pour l'équation.
import yfinance as yf
import numpy as np

def get_market_data(ticker="SPY", period="2y"):
    data = yf.download(ticker, period=period)
    # Calcul log returns
    data['Log_Ret'] = np.log(data['Close'] / data['Close'].shift(1))
    # Volatilité roulante (proxy pour la masse)
    data['Volatility'] = data['Log_Ret'].rolling(window=20).std()
    return data.dropna()

Module 2 : Construction du Potentiel V(x)

Le potentiel $V(x)$ représente le "paysage" dans lequel le prix évolue. Les niveaux de support et résistance créent des "murs".

Stratégie :

  • Identifier les pics de volume (Volume Profile).
  • Utiliser les moyennes mobiles ou les Bandes de Bollinger comme puits de potentiel quadratique.
  • Inverser la densité de probabilité historique : $V(x) \propto - \ln(P_{hist}(x))$.

Module 3 : Solveur Numérique (FDM)

Résolution de l'équation de Schrödinger indépendante du temps (TISE) ou dépendante (TDSE) via la méthode des différences finies.

from scipy.sparse import diags
from scipy.sparse.linalg import eigs

def solve_schrodinger(potential, mass=1.0, dx=0.1):
    N = len(potential)

    # Construction de l'Hamiltonien (Matrice Tridiagonale)
    # Terme cinétique (-h^2 / 2m * d^2/dx^2)
    k = -1 / (2 * mass * dx**2)
    main_diag = potential + 2/ (2 * mass * dx**2)
    off_diag = k * np.ones(N-1)

    H = diags([off_diag, main_diag, off_diag], [-1, 0, 1])

    # Calcul des valeurs propres (Énergies) et vecteurs propres (Psi)
    energies, wavefunctions = eigs(H, k=5, which='SR') # 'SR' = Smallest Real part

    return energies, wavefunctions

Module 4 : Moteur de Signal (Effet Tunnel)

Génération des ordres d'achat/vente basés sur le coefficient de transmission $T$.

Signal ACHAT (Long)

Déclenché si le prix est sous une résistance MAIS que la probabilité de tunnel $T > 50\%$ et que le momentum quantique est positif.

Signal VENTE (Short)

Déclenché si le prix est sur un support MAIS que la probabilité de chute (tunnel vers le bas) est élevée.

def generate_signal(transmission_prob, trend_momentum):
    THRESHOLD = 0.65

    if transmission_prob > THRESHOLD and trend_momentum > 0:
        return "BUY_BREAKOUT"
    elif transmission_prob > THRESHOLD and trend_momentum < 0:
        return "SELL_BREAKDOWN"
    else:
        return "HOLD"

import numpy as np
import pandas as pd
import scipy.sparse as sparse
from scipy.sparse.linalg import eigs

class QuantumFinanceBot:
    def __init__(self, ticker="SPY"):
        self.ticker = ticker
        self.hbar = 1.0  # Planck constant normalisée
        self.mass = 1.0  # Masse effective par défaut

    def prepare_data(self, prices):
        """Transforme les prix en log-returns et normalise"""
        log_ret = np.log(prices / prices.shift(1)).dropna()
        self.volatility = log_ret.rolling(20).std().iloc[-1]
        # La masse est inversement proportionnelle à la volatilité²
        self.mass = 1.0 / (self.volatility ** 2)
        return log_ret

    def build_hamiltonian(self, potential_barrier, n_points=100):
        """Construit la matrice Hamiltonienne H = T + V"""
        dx = 1.0 / n_points
        diag_potential = potential_barrier

        # Terme cinétique (Laplacien discret)
        # -h² / 2m * (d²psi / dx²)
        kinetic_factor = - (self.hbar**2) / (2 * self.mass * dx**2)

        # Diagonale principale : V(x) - 2 * facteur cinétique
        main_diag = diag_potential - 2 * kinetic_factor

        # Diagonales secondaires : facteur cinétique
        off_diag = kinetic_factor * np.ones(n_points - 1)

        # Matrice clairsemée
        H = sparse.diags([off_diag, main_diag, off_diag], [-1, 0, 1])
        return H

    def calculate_transmission(self, energy, barrier_height, width):
        """Formule analytique de l'effet tunnel pour barrière rectangulaire"""
        # T approx = exp(-2 * K * L)
        # K = sqrt(2m(V-E)) / hbar
        if energy > barrier_height:
            return 1.0 # Transmission classique

        kappa = np.sqrt(2 * self.mass * (barrier_height - energy)) / self.hbar
        transmission = np.exp(-2 * kappa * width)
        return transmission

    def analyze_market(self, current_price, resistance_level, market_momentum):
        """Fonction principale de décision"""
        # Énergie cinétique du marché ~ momentum²
        energy = market_momentum ** 2

        # Hauteur de barrière ~ (Résistance - Prix Actuel)
        barrier_height = (resistance_level - current_price) * 10

        prob = self.calculate_transmission(energy, barrier_height, width=0.5)

        print(f"Probabilité de Tunneling (Breakout) : {prob:.2%}")

        if prob > 0.6:
            return "SIGNAL: ACHAT (Breakout Imminent)"
        elif prob < 0.1:
            return "SIGNAL: REJET (Rebond Baissier)"
        else:
            return "SIGNAL: NEUTRE"
